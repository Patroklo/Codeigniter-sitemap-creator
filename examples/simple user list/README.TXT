SPANISH

En este ejemplo crearemos un sitemap muy sencillo que apunta a las páginas públicas de todos los usuarios de una página web. Para ello crearemos una lista en el helper llamada "userList" que heredará de la "urlList" principal. Después cargaremos la lista completa de todos los usuarios que tengamos en la base de datos. Posteriormente formaremos a través del campo select de la query la dirección de la página web y llamaremos al sitemap builder.

Para lanzar el generador de sitemaps en este ejemplo sólo hay que acceder al index del controller "launcher".

ENGLISH


In this example we will make a very simple sitemap that points to the public url of every user of a web. For that we will make a new list in the helper called "userList" that will extend from the main "urlList". Then we will load the user list from the database. After that we'll make from the select of the query the users public url and call the sitemap builder.



For launching the sitemap generator you only have to access the index of the "launcher" controller.
